# Hub v0.3 — Rediseño principal (hub_main_v5)

Referencia visual: `Vantare-Overlays/hub_main_v5.html`

## Objetivo

Reemplazar el shell actual (`HubSidebar` + `HubHeader` + páginas utilitarias) por un **hub tipo plataforma**: topbar fija, hero cinematográfico, grid de cards y sensación premium Vantare (negro + rojo vino).

La lógica de Ingeniero (WS, PTT, spotter, config) **no se reescribe**; se reubica en el nuevo layout.

---

## Mapa HTML → React

| Bloque v5 | Componente propuesto | Datos reales v0.3 |
|-----------|---------------------|-------------------|
| Top nav (logo, Hub/Overlays/Telemetría/Setup, live) | `HubTopNav.tsx` | Live = `backendHealth` + LMU; Setup → drawer config |
| Hero VANTARE + coche/circuito | `HubHero.tsx` | LMU API / telemetría: coche, pista, sesión |
| Banner próximo evento | `NextEventBanner.tsx` | **Mock v0.3** (CMS/manual JSON); real en v0.4 |
| Ratings piloto / seguridad | `RatingCards.tsx` | **Mock** o oculto hasta integración iRacing/LMU stats |
| Chart iRating | `RatingChart.tsx` | Chart.js o `recharts`; historial local / mock |
| Carreras recientes | `RecentSessionsList.tsx` | `HistorialPage` + traces/sessions persistidas |
| Vantare Pro | `ProUpsellCard.tsx` | Estático / enlace externo |
| Ecosistema Vantare | `EcosystemPanel.tsx` | Links: Ingeniero (activo), Telemetría (futuro) |

## Design tokens (Tailwind v4)

Portar desde el HTML:

- Fuentes: `Inter` (body), `Rajdhani` (display), `Space Mono` (datos)
- Colores: `vantare-bg #080808`, `surface #0F0F0F`, `red-500 #C1121F`, etc.
- Utilidades: `.glass-panel`, `.card-sleek`, `.btn-primary`, `.premium-bg` → `@layer components` en `hub-v3.css`

No usar CDN Tailwind en prod; integrar tokens en `frontend/src` existente (ya hay Tailwind v4).

## Riesgos

| Riesgo | Mitigación |
|--------|------------|
| Chart.js peso bundle | `recharts` tree-shaken o lazy `import('chart.js')` |
| ConfigTab monolítico | Extraer secciones a panels reutilizables antes de drawer |
| Mock data confunde usuarios | Badge “Próximamente” en ratings/evento hasta datos reales |
