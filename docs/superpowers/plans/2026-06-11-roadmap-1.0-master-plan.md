# Roadmap 1.0 — Plan maestro (paridad CrewChief, arquitectura simple)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement hito-by-hito.  
> **Orquestador (obligatorio):** [`2026-06-11-roadmap-1.0-ORCHESTRATOR-INDEX.md`](2026-06-11-roadmap-1.0-ORCHESTRATOR-INDEX.md)  
> **Visión producto:** [`../../ROADMAP-1.0.md`](../../ROADMAP-1.0.md)  
> **Base estable:** v0.2.14 Voice Beta (LMU)

**Goal:** Llevar Vantare de v0.2.14 a **1.1** imitando y superando CrewChief en voz/ingeniero/spotter/pit — **monolito simple**, sin overlays telemetría in-game, apps Go y Vantare **independientes** hasta 1.1.

**Architecture:** Mantener ADR-004-R1 (`backend.exe` = race_loop + voice_loop + CC). Cada versión 0.3–0.9 extiende el monolito LMU. **1.0** añade clonación TTS en el mismo proceso. **1.1** añade mapper iRacing en `shared-telemetry` + Suite opcional (Go ↔ Vantare sin acoplamiento runtime obligatorio).

**Tech Stack:** Python 3.12, FastAPI, Electron, PyInstaller, pygame voice, Edge/Gemini TTS, Rust `shared-telemetry` (solo 1.1), Go overlay (repo externo, 1.1).
